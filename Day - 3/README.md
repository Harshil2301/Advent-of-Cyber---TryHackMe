## üß† Key Concepts Covered

### üîπ What Are Logs?
Logs are records of events generated by:
- Operating systems  
- Applications  
- Network devices  
- Security tools  

They provide visibility into **what happened, when it happened, and where it happened**.

---

### üîπ Security Events
A **security event** is a meaningful occurrence derived from logs, such as:
- Failed login attempts  
- Privilege escalation  
- Unusual network activity  

Events help analysts identify suspicious or malicious behavior.

---

### üîπ SIEM (Security Information and Event Management)
A **SIEM** system collects logs from multiple sources, normalizes them, and correlates events to detect potential threats.

Core SIEM functions include:
- Centralized log collection  
- Event correlation  
- Alert generation  
- Incident investigation support

## üîç SOC Analyst Perspective
From a SOC standpoint, SIEM tools help analysts:
- Detect anomalies across large datasets  
- Prioritize alerts based on severity  
- Investigate incidents efficiently  
- Reduce response time through correlation  

Without SIEM, monitoring modern infrastructures at scale becomes impractical.

### Q1Ô∏è‚É£ What is a log in cybersecurity?
**Answer:**  
A log is a record of system, application, or network activity that provides information about events occurring within an environment.

---

### Q2Ô∏è‚É£ What is the difference between logs and security events?
**Answer:**  
Logs are raw records of activity, while security events are meaningful or suspicious occurrences identified from logs through analysis or correlation.

---

### Q3Ô∏è‚É£ What is a SIEM system?
**Answer:**  
A SIEM system is a security solution that centralizes log collection, correlates events, and generates alerts to help detect and respond to security incidents.

---

### Q4Ô∏è‚É£ Why is log correlation important?
**Answer:**  
Log correlation helps identify patterns across multiple systems that may indicate an attack, which might not be visible when logs are viewed individually.

---

### Q5Ô∏è‚É£ How does SIEM support SOC operations?
**Answer:**  
SIEM enables SOC teams to monitor environments continuously, prioritize alerts, investigate incidents, and respond to threats more efficiently.
